# WGU Companion

WGU Companion is an Android mobile application that helps WGU students manage their academic terms, courses, and assessments. All data is stored locally on the device using a SQLite database.

## Features

- Secure PIN login
- Add, edit, and delete terms, courses, and assessments
- Date notifications for course and assessment start and end dates
- Instructor contact info stored per course
- Course notes with sharing support via email or SMS
- Search courses and generate reports by keyword

## Download

Visit the hosted page to download the latest APK:

https://z-cabric.github.io/wgu-companion/

## Installation

Download the APK on an Android device. Open the downloaded file and allow installation from unknown sources if prompted. This warning appears for any app installed outside the Google Play Store. After installation, open WGU Companion and create a PIN to get started.

## Built With

- .NET MAUI
- C#
- XAML
- SQLite
