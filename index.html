WGU Companion

WGU Companion is an Android mobile application that helps WGU students manage their academic terms, courses, and assessments. The app stores all data locally on the device using a SQLite database and supports adding, editing, deleting, searching, and reporting on academic records.

Features

Term, course, and assessment management. Local notifications for course and assessment dates. Note sharing through email or SMS. Search and report generation. Secure PIN login.

Download WGU Companion APK
Installation: Download the APK on an Android device. Open the downloaded file and allow installation from unknown sources if prompted. After installation, open the WGU Companion app and create a PIN to begin.

Built with .NET MAUI, C#, and XAML. Developed for WGU course D424 Software Engineering Capstone.
